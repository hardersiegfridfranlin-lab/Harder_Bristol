💎 Diamond Rush: Professional Edition
🎮 Game Title & Description

Diamond Rush: Professional Edition is a Java Swing-based idle clicker game where players mine gems by clicking a central diamond and upgrading their mining efficiency.

Players can:

Click the diamond to earn gems
Buy upgrades to increase click power and passive income
Unlock Gem Multipliers (Iron → Diamond) to boost click value
Purchase Auto Miner Skins (Shovel → Jackhammer) to speed up passive generation
Progress through a scaling economy with increasing costs and rewards

The game includes smooth animations, floating text effects, screen shake feedback, and an auto-save system.

⚙️ Setup Instructions (How to Run the Code)
Requirements:
Java JDK 8 or higher
Any Java IDE (e.g., IntelliJ, Eclipse, NetBeans) or terminal
Steps:

Copy the code into a file named:

DiamondRush.java
Open terminal or command prompt in the file directory

Compile the program:

javac DiamondRush.java

Run the game:

java DiamondRush

✅ The game window will open automatically.

🎯 List of Controls
Action	Control
Mine gems	Left-click on the diamond
Buy upgrades	Click buttons on the right panel
Scroll shop	Mouse wheel
Reset game	"Reset Progress" button (confirmation required)
🛠️ Game Features
⛏ Upgrades
Auto Miner – Generates passive income
Build Mine – Adds additional passive production
Sharpen Pickaxe – Increases click power
💎 Gem Shop (Click Multipliers)

Unlock in order:

Iron (x1.5)
Gold (x2.0)
Ruby (x2.5)
Sapphire (x3.0)
Emerald (x4.0)
Diamond (x5.0)

⚠️ Buying a gem resets your current gems to 0

🪄 Auto Miner Skins (Speed Multipliers)

Unlock in order:

Shovel (x2)
Pickaxe (x4)
Jackhammer (x7)
✨ Visual Effects
Floating "+gems" text on click
Screen shake feedback
Pulsing animated diamond
Dynamic color changes based on upgrades
💾 Auto Save System
Game automatically saves every 5 seconds
